
# Responsive Blog Webpage with API Integration

## Project Overview

This project is a responsive blog webpage built using HTML, CSS, and JavaScript. It fetches blog posts from the JSONPlaceholder API and displays the latest six posts with proper loading and error handling.

## Features

- Responsive design using CSS Grid
- Semantic HTML structure
- External CSS and JavaScript
- Fetch API integration
- Displays latest 6 blog posts
- Loading state while fetching data
- Error state if API request fails

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

## API Used

https://jsonplaceholder.typicode.com/posts

## How to Run

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Install the Live Server extension (if not installed).
4. Right-click on `index.html`.
5. Select **Open with Live Server**.
6. The webpage will load and display the latest six blog posts.

## Output

- Displays latest 6 blog posts
- Responsive layout
- Loading message while data is fetched
- Error message if the API request fails
